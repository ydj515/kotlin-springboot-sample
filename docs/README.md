# 문서 안내

## 개요

이 디렉토리는 현재 JPA Kotlin Spring Boot 샘플의 구조, 계층 경계, 실행 방법, JPA 예제 포인트를 주제별로 정리한 문서 모음입니다.

## 이 문서를 보면 좋은 경우

- 저장소를 처음 열고 패키지 구조를 빠르게 파악하고 싶을 때
- `UseCase / command / result / policy` 규칙이 실제로 어떻게 적용되는지 알고 싶을 때
- `Post`, `User`, `Order` 도메인에서 어떤 JPA 예제를 보여주는지 한 번에 보고 싶을 때
- 새 도메인을 추가할 때 어떤 계층에 어떤 파일을 둬야 할지 기준이 필요할 때
